# marrow-privacy

This repository hosts the static privacy policy and terms of service for **Marrow (Dog Wellness)** by Tritan, LLC.

It is deployed publicly via **GitHub Pages** to satisfy app store and marketplace requirements.

*   **URL**: [https://kylemillerbuilds.github.io/marrow-privacy/privacy.html](https://kylemillerbuilds.github.io/marrow-privacy/privacy.html)
*   **Contents**:
    *   `index.html` — Automated redirect to the policy page.
    *   `privacy.html` — The privacy policy document.
